# JSONSchema.xyz

This is a collection of JSON Schema documents for ASP.NET Core. The schemas are used to validate JSON documents that are used in the ASP.NET Core framework.  It contains schemas for the following:

- [DownstreamApisConfiguration.schema](https://jsonschema.xyz/schemas/DownstreamApisConfiguration.schema)
- [JsonSerializerOptions.schema](https://jsonschema.xyz/schemas/JsonSerializerOptions.schema)
- [MicrosoftIdentityOptions.schema](https://jsonschema.xyz/schemas/MicrosoftIdentityOptions.schema)
- [MicrosoftIdentityWebApplicationOptions.schema](https://jsonschema.xyz/schemas/MicrosoftIdentityWebApplicationOptions.schema)
- [OpenApiInfo.schema](https://jsonschema.xyz/schemas/OpenApiInfo.schema)
- [OpenIdConnectOptions.schema](https://jsonschema.xyz/schemas/OpenIdConnectOptions.schema)
- [Telegram.json.schema](https://jsonschema.xyz/schemas/Telegram.json.schema)
- [VerifiableCredentialsDownstreamApiConfiguration.schema](https://jsonschema.xyz/schemas/VerifiableCredentialsDownstreamApiConfiguration.schema)
- [dotnet-tools.json.schema](https://jsonschema.xyz/schemas/dotnet-tools.json.schema)
- [global.json.schema](https://jsonschema.xyz/schemas/global.json.schema)
